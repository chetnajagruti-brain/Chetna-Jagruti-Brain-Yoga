TROUBLESHOOTING:
-----------------

Image not showing?
  → Check filename: bg-1.jpg, bg-2.jpg, bg-3.jpg (exact match required)
  → Verify location: c:\Brain yoga\new-site\assets\images\
  → Clear browser cache (Ctrl+Shift+Delete)
  → Check file extensions are lowercase

Can't read text?
  → Dark overlay is 50% opacity - this is intentional for readability
  → To adjust: Edit styles.css line ~708
  → Change rgba(0, 0, 0, 0.50) to darker (0.70) or lighter (0.30)

Pages loading slow?
  → Optimize images (TinyPNG.com or similar)
  → Reduce to 300-400KB per image
  → Keep 16:9 aspect ratio
